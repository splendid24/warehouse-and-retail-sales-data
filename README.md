# 📊 Sales Data Analysis Project

## 📌 Description
This project analyzes a sales dataset containing information about suppliers, items, and different types of sales over time.  
The dataset tracks monthly sales performance, including retail sales, warehouse sales, and transfers.

The goal of this project is to:
- Understand sales trends over time
- Analyze supplier and product performance
- Identify patterns in retail vs warehouse sales
- Support data-driven decision making

---

## 📂 Dataset Overview

The dataset contains the following columns:

- *YEAR* – Year of the sales record  
- *MONTH* – Month of the sales record  
- *SUPPLIER* – Name of the supplier  
- *ITEM CODE* – Unique identifier for each product  
- *ITEM DESCRIPTION* – Description of the product  
- *ITEM TYPE* – Category/type of the item  
- *RETAIL SALES* – Sales made directly to customers  
- *RETAIL TRANSFERS* – Transfers between retail outlets  
- *WAREHOUSE SALES* – Sales made from warehouse distribution  
- *TOTAL SALES* – Total of all sales types
